# ✅ إصلاح حسابات الديون والمستحقات في الداشبورد

## 🔧 المشاكل التي تم إصلاحها:

### 1. المشكلة الأولى: الحسابات كانت صفر
**السبب:** الـ `balance` و `net` مش موجودين في الـ models، لازم يتحسبوا من الـ services

**الحل:**
- استخدام الـ services لحساب الأرصدة الفعلية:
  - `ClientService.computeClientTotals()` - لحساب أرصدة العملاء
  - `CrusherService.computeCrusherTotals()` - لحساب أرصدة الكسارات
  - `ContractorService.computeContractorTotals()` - لحساب أرصدة المقاولين
  - `SupplierService.computeSupplierTotals()` - لحساب أرصدة الموردين
  - `PayrollService.calculateEmployeeBalance()` - لحساب أرصدة الموظفين

### 2. المشكلة الثانية: ترتيب المربعات
**المطلوب:** المربعات تظهر بعد صافي الربح مباشرة

**الحل:**
- نقل المربعين بعد مربع "صافي الربح" مباشرة
- الترتيب الجديد:
  1. إجمالي المبيعات
  2. إجمالي التكاليف
  3. صافي الربح
  4. **إجمالي الديون** (جديد)
  5. **إجمالي المستحقات** (جديد)
  6. العملاء
  7. المقاولين
  8. الكسارات
  9. التسليمات
  10. الموظفين

### 3. تحسين التسميات
- **"إجمالي الديون اللي ليا"** → **"إجمالي الديون"** (مستحقات لنا)
- **"إجمالي الديون اللي عليا"** → **"إجمالي المستحقات"** (مستحقات علينا)

---

## 📊 الحسابات الصحيحة:

### إجمالي الديون (Receivables - مستحقات لنا):
```javascript
totalReceivables = 
  العملاء (balance > 0) +           // العملاء المديونين لنا
  الكسارات (net < 0) +              // الكسارات المديونة لنا
  المقاولين (balance < 0) +         // المقاولين المديونين لنا
  الموردين (balance < 0) +          // الموردين المديونين لنا
  الموظفين (balance > 0)            // الموظفين المديونين لنا
```

### إجمالي المستحقات (Payables - مستحقات علينا):
```javascript
totalPayables = 
  الموردين (balance > 0) +          // مستحقات للموردين
  الكسارات (net > 0) +              // مستحقات للكسارات
  المقاولين (balance > 0) +         // مستحقات للمقاولين
  الموظفين (balance < 0) +          // مستحقات للموظفين
  العملاء (balance < 0)             // مستحقات للعملاء
```

---

## 🎯 المنطق المحاسبي:

### الديون (Receivables):
- **موجب** = الطرف الآخر مديون لنا
- **سالب** = احنا مديونين للطرف الآخر

### القاعدة:
- **العملاء:** balance موجب = مديونين لنا
- **الكسارات:** net سالب = مديونة لنا
- **المقاولين:** balance سالب = مديونين لنا
- **الموردين:** balance سالب = مديونين لنا
- **الموظفين:** balance موجب = مديونين لنا
